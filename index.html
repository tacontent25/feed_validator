XML Валидатор

На этой странице вы можете проверить свой XML файл на соответствие требованиям для передачи актуальной информации на TrendAgent.
Важно! Валидатор проверяет наличие необходимых тегов но не проверяет их содержимое, поэтому даже при валидном XML, фид может не подойти из-за некорректного наполнения. Это выявляется только при ручной проверке перед обработкой источника.
